# 修仙乙女游戏剧情设计文档

## 基础设定

### 时间线
- 0-5岁：凡人童年（可能遇到青梅竹马）
- 6岁：测灵根，入宗门
- 6-18岁：少年期（练气期）
- 18-50岁：青年期（筑基期）
- 50-150岁：壮年期（金丹期）
- 150-500岁：中年期（元婴期）
- 500岁+：后期（化神/渡劫/飞升）

### 属性
- 体魄（0-100）
- 根骨（0-100）
- 容貌（0-100）
- 机缘（0-100）
- 修为（0-100，满了可突破）

### 境界与寿命（简化版）
| 境界 | 寿命上限 |
|------|---------|
| 凡人 | 60岁 |
| 练气 | 80岁 |
| 筑基 | 100岁 |
| 金丹 | 130岁 |
| 元婴 | 160岁 |
| 飞升 | 通关 |

### 妖族设定
所有妖族修炼到一定境界即可化为人形，外表与人无异。
只有在动用妖力或情绪激动时才会显露部分本体特征（如耳朵、尾巴、眼瞳变色等）。
妖族可以是任何身份：路人、朋友、道侣、师父等。

### 货币
灵石

### 关系类型
- 父母
- 青梅竹马
- 师父
- 师兄/师弟/师姐/师妹
- 同门
- 道侣
- 知己/挚友
- 其他宗门的人（友/敌）
- 妖族
- 仇人

### NPC属性
每个NPC都有：
- 名字（随机生成）
- 性别
- 年龄（会增长，会死）
- 外貌描述（随机）
- 性格（随机）
- 境界（会提升，会陨落）
- 好感度（-100到100）
- 身份（师父/师兄/敌宗弟子/妖族...）
- 状态（正常/受伤/闭关/失踪/死亡）

## 宗门设定

### 正道宗门
1. **青云宗** - 正道第一，规矩严格
2. **天剑门** - 以剑入道，剑修为主
3. **落霞派** - 女修为主，功法阴柔
4. **百草谷** - 擅长丹药，医修聚集
5. **万法宗** - 功法众多，杂而不精

### 邪道宗门
1. **幽冥教** - 修炼诡异，手段狠辣
2. **血煞门** - 以血入道，凶名远扬
3. **毒蛊宗** - 用毒高手，防不胜防

### 妖族势力
1. **万妖山** - 妖族大本营
2. **狐族** - 最擅惑人
3. **龙族** - 最为强大
4. **凤族** - 高贵傲气

## 关系变化规则

### 好感度影响
- 好感度 < -50：仇人
- 好感度 -50 ~ 0：陌生/敌意
- 好感度 0 ~ 30：普通
- 好感度 30 ~ 60：友好
- 好感度 60 ~ 80：亲密
- 好感度 > 80：知己/可发展道侣

### NPC死亡条件
1. 剧情死亡（固定事件）
2. 寿命耗尽（根据境界）
3. 突破失败
4. 战斗死亡
5. 随机意外（小概率）

### 关系升级
- 普通 → 朋友：好感度 > 30 + 相处时间
- 朋友 → 知己：好感度 > 60 + 重要事件
- 任意 → 道侣：好感度 > 80 + 表白事件
- 师徒：固定分配或拜师事件

## 技术备注

### 每年事件触发规则
1. **不是每年都有事件**：每年有 60-70% 概率触发随机事件
2. **伏笔优先**：如果有待触发的伏笔/后续剧情，强制触发（100%概率）
3. **平淡年份**：没有事件时，显示简单的日常描述（修炼、采药、平静的一年）

### 伏笔/剧情链系统
用**标记（flag）**实现剧情关联：

**标记类型：**
- `met_xxx`：遇见过某人（如 met_mysterious_white_man）
- `has_xxx`：拥有某物品（如 has_keepsake_from_师兄名）
- `event_xxx`：触发过某事件（如 event_childhood_separation）
- `saved_xxx`：救过某人
- `enemy_xxx`：与某人结仇

**后续事件触发条件举例：**
```
【事件：神秘白衣人再现】
条件：has_flag("met_mysterious_white_man") AND 年龄 >= 18
优先级：高（伏笔类）
```

**伏笔示例：**
| 前置事件 | 获得标记 | 后续事件 | 触发时机 |
|---------|---------|---------|---------|
| 神秘白衣人路过 | met_mysterious_white_man | 白衣人揭露身份 | 15-25岁 |
| 青梅竹马分离 | event_childhood_separation | 重逢 | 入宗门后随机 |
| 获得定情信物 | has_keepsake_xxx | 信物相关剧情 | 该人出事/重逢时 |
| 救了妖族 | saved_妖族名 | 妖族报恩 | 你遇险时 |
| 与某人结仇 | enemy_xxx | 仇人复仇 | 10-30年后 |
| 发现神秘印记 | has_mysterious_mark | 印记觉醒 | 突破/危机时 |

### 随机事件触发条件
每个事件都有触发条件：
- 年龄范围
- 境界要求
- 是否有某关系
- 某人好感度
- 是否有某标记（伏笔系统）
- 前置事件是否触发过

### 事件优先级
1. **强制事件**：伏笔后续、关键剧情节点（100%触发）
2. **高优先级**：重要关系人相关事件
3. **普通事件**：随机日常
4. **低优先级**：已触发过的类似事件

### 事件权重
- 基础权重
- 根据属性调整
- 根据已有关系调整
- 有伏笔标记的后续事件权重大幅提升
- 避免重复（已触发的降低权重）

- 玩家所有属性
- 所有关系人及其属性
- 已触发事件列表
- 各种标记/flag
- 物品列表
